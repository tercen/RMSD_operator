# RMSD operator

##### Description

`RMSD` operator returns the Relative Mean Square Deviation of two variables.

##### Usage

Input projection|.
---|---
`x-axis`        | numeric, input data, per cell 
`y-axis`        | numeric, input data, per cell 

Output relations|.
---|---
`rmsd`        | numeric, RMSD of the input data
